Installation

Windows
Linux
macOS

Windows Installation

  1. Download byc-windows.zip
  2. Extract the zip file
  3. Double-click byc.exe to start

Linux Installation

  1. Download byc-linux.tar.gz
  2. Extract: tar -xzf byc-linux.tar.gz
  3. Make executable: chmod +x byc
  4. Run: ./byc

macOS Installation

  1. Download byc-macos.tar.gz
  2. Extract: tar -xzf byc-macos.tar.gz
  3. Make executable: chmod +x byc-mac
  4. Run: ./byc-mac

Coin System

Golden Block Coins

  • Leah (base unit) Mineable
  • Shiblum Mineable
  • Shiblon Mineable
  • Antium Cross-Block
  • Senine
  • Seon
  • Shum
  • Limnah
  • Antion

Silver Block Coins

  • Leah (base unit) Mineable
  • Shiblum Mineable
  • Shiblon Mineable
  • Senum
  • Amnor
  • Ezrom
  • Onti

Special Coins

  • Ephraim Coin Max: 15M

    Creation Requirements (Fibonacci Sequence):

    • 1 Leah
    • 1 Shiblum
    • 2 Shiblon
    • 3 Senine
    • 5 Seon
    • 8 Shum
    • 13 Limnah
    • 21 Antion

    Total Value: 54 Leah

  • Manasseh Coin Max: 15M

    Creation Requirements (Fibonacci Sequence):

    • 1 Senum
    • 1 Amnor
    • 2 Ezrom
    • 3 Onti
    • 1 Antion (from Golden Block)

    Total Value: 28 Leah

  • Joseph Coin Max: 3M

    Creation Requirements:

    • 1 Ephraim Coin
    • 1 Manasseh Coin

Note: Only Leah, Shiblum, and Shiblon are mineable coins, found in both Golden and Silver blocks. All other coins are obtained through different mechanisms. An Antium is unique as it can be transferred between Golden and Silver blocks, while other coins are restricted to their respective blocks.

Special Coins Note: Ephraim and Manasseh coins are created using Fibonacci sequence amounts of their respective block coins. Once you have both Ephraim and Manasseh coins, you can create Joseph coins. Each special coin has a maximum supply limit.
